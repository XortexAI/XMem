## 2026-04-05 - Concurrent multi-repo search\n**Learning:** Sequential async operations in loops (e.g., `for repo in repos: await search(repo)`) create significant linear latency bottlenecks in retrieval pipelines.\n**Action:** Use `asyncio.gather` combined with list flattening to parallelize these calls, reducing multi-repo search latency to the duration of the single slowest namespace query.
